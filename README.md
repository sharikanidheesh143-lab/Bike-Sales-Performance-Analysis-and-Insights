Bike Sales Performance Analysis and Insights

This project analyzes bike sales data using Excel (Phase 1) and Power BI (Phase 2).  
It focuses on cleaning the dataset, preparing it for analysis, and building an interactive dashboard to understand buying behavior, income influence, age trends, and loan eligibility.

Phase 1 – Excel (Data Cleaning & Preparation)
Phase 1 involves preparing the raw bike sales dataset for analysis.

✔ Tasks Completed in Excel:
- Removed duplicates and corrected inconsistencies  
- Standardized columns (names, data types, date formats)  
- Handled missing values  
- Created calculated fields such as:  
  - Total Sales  
  - Income Category  
  - Age Group  
  - Loan Eligibility.
   
Phase 2 – Power BI (Dashboard & Insights)
Phase 2 visualizes the cleaned data to identify important patterns in bike purchasing behavior.
 ✔ Power BI Dashboard Includes:
- Loan Eligibility by Income & Education  
- Age trend of bike buyers  
- Bike Purchase by Income Category  
- Eligibility Count by Income  
- Purchase vs Target Gauge  
- Interactive filters for:
  - Education  
  - Gender  
  - Marital Status  
  - Income Category  
  - Loan Eligibility
 
Key Insights
 -Higher-income individuals show higher loan eligibility  
- Younger age groups purchase more bikes  
- Education level influences loan approval  
- Taxable income category shows more purchasing activity

  Summary
This project demonstrates end‑to‑end data analysis:
1. Cleaning and preparing raw data (Excel)  
2. Visualizing insights (Power BI)  
3. Understanding customer behavior and purchase patterns
